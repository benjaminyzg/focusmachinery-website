# Focus Machinery Portfolio

A high-performance, responsive portfolio web application designed for a modern industrial engineering brand. This platform highlights specialized fluid engineering setups, technical services, and advanced purification systems.

## 🛠️ Tech Stack

- **Frontend:** React, React Router, TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Backend Environment:** Python (Django Management Tools)

## 🚀 Features

- **Responsive Architecture:** Clean 4-column dynamic layouts optimized for desktop, tablet, and mobile screens.
- **Modern UI/UX:** Styled completely with utility-first Tailwind CSS for lightning-fast UI adjustments and seamless styling.
- **Type Safety:** Built with TypeScript to ensure strict typing and robust component properties across routes.
